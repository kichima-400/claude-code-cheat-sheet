# Claude Code チートシート (日本語)

Claude Code の使い方を1ページにまとめた日本語チートシートです。

> **情報の基準日**: 2026/04/02 時点の公式ドキュメントをもとに作成

## 内容

| カード | 内容 |
|--------|------|
| キーボードショートカット | 基本操作・テキスト編集・モード切替・マルチライン入力 |
| スラッシュコマンド | セッション・設定・ツール・ユーティリティ系コマンド |
| CLI コマンドとフラグ | `claude` コマンドのオプション一覧 |
| ワークフローとヒント | プランモード・思考・Git Worktree・SDK/ヘッドレス実行 |
| フック設定 | フックイベント・タイプ・終了コード・設定例 |
| スキル | ビルトインスキル・カスタムスキルの作り方 |
| 設定ファイル | settings.json・CLAUDE.md の場所と主要キー |
| パーミッションモードとモデル | モード一覧・モデル ID・エフォートレベル・MCP |

## 表示

**[チートシートを開く](https://htmlpreview.github.io/?https://github.com/kichima-400/claude-code-cheat-sheet/blob/master/cheatsheet.html)**

またはローカルで `cheatsheet.html` をブラウザで開いても使えます。レスポンシブ対応（4列 / 2列 / 1列）。

## 情報源

すべての内容は [Anthropic 公式ドキュメント](https://docs.anthropic.com/ja/docs/claude-code/overview) をもとに作成しています。
